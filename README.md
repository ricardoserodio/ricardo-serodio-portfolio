# Ricardo Serôdio – Professional Portfolio Website

Professional portfolio website for Ricardo Serôdio, focused on banking operations, wealth management, financial data quality, banking analytics, AI finance and business operations.

This website is designed as a public professional landing page for international recruiters, online applications, LinkedIn, GitHub, CV references and professional outreach.

## Core Narrative

This website is the central point of Ricardo Serôdio's professional narrative:

```text
Banking experience + financial data quality + banking analytics + public portfolio + international career focus.
```

The goal is to position Ricardo as a finance and banking professional combining real experience in regulated financial environments with data quality awareness, banking analytics, AI-assisted finance workflows and a public GitHub portfolio.
